# MacDraw 使用手冊

MacDraw 是一個 macOS 桌面覆蓋繪圖工具，可以直接在桌面與其他視窗上方進行標記。

## 1. 功能簡介

- 透明覆蓋層，可直接畫在螢幕最上層
- 支援多螢幕
- 支援 `自由手繪` 與 `圓角矩形`
- 支援 `畫筆` 與 `橡皮擦`
- 可調整顏色與筆刷粗細
- 支援 `Undo` 與 `Clear`
- 支援 `Trail Effect` 軌跡消失效果
- 按 `Escape` 可立即停止繪圖

## 2. 目前預設設定

啟動後的預設狀態如下：

- 形狀模式：`Rectangle`
- 工具模式：`Pen`
- 軌跡效果：`Enable`
- 軌跡消失時間：`0.5 秒`

也就是說，程式一開啟後，直接拖曳滑鼠就會畫出會快速消失的圓角矩形。

## 3. 建置方式

在專案資料夾中執行：

```bash
./scripts/build-app.sh
```

建置完成後會產生：

- 可直接執行的程式：`build/bin/MacDraw`
- App Bundle：`build/MacDraw.app`

## 4. 啟動方式

目前建議直接執行二進位檔：

```bash
./build/bin/MacDraw
```

雖然建置流程也會產生 `.app`，但在目前這個環境中，macOS 可能會拒絕直接開啟手動組裝的 `.app`。因此以 `build/bin/MacDraw` 啟動最穩定。

## 5. 操作面板說明

程式啟動後，會看到一個浮動控制面板 `MacDraw Controls`。

### Shape

- `Freehand`：自由手繪
- `Rectangle`：畫圓角矩形

### Tool

- `Pen`：正常繪圖
- `Eraser`：擦除已畫出的內容

### Color

- 設定線條顏色

### Brush Size

- 設定線條粗細

### Trail Effect

- `Enable Trail Effect`：
  開啟後，新畫出的圖形會逐步消失
- `Trail Disappear Time`：
  控制圖形從出現到完全消失所需的時間

## 6. 如何畫圖

### 畫圓角矩形

1. 在 `Shape` 選擇 `Rectangle`
2. 在 `Tool` 選擇 `Pen`
3. 按住滑鼠左鍵拖曳
4. 放開滑鼠後完成矩形

### 自由手繪

1. 在 `Shape` 選擇 `Freehand`
2. 在 `Tool` 選擇 `Pen`
3. 按住滑鼠左鍵直接書寫或塗畫

### 擦除內容

1. 在 `Tool` 選擇 `Eraser`
2. 於畫面上拖曳即可擦除

## 7. 常用操作

### 停止繪圖

- 按 `Escape`
- 或按控制面板中的 `Disable Drawing`

停止後，覆蓋層會改成不攔截滑鼠點擊，你可以直接操作桌面或其他 App。

### 恢復繪圖

- 按控制面板中的 `Enable Drawing`

### 復原上一筆

- 按 `Undo`

### 清除全部

- 按 `Clear`

## 8. 多螢幕支援

如果你有多個顯示器，MacDraw 會在每個螢幕上建立透明覆蓋層。

若螢幕配置改變，例如：

- 接上新螢幕
- 拔除外接螢幕
- 更改顯示器排列方式

程式會自動重建覆蓋層。

## 9. 注意事項

- 控制面板會保持在覆蓋層上方，方便你隨時切換模式
- 若開啟 `Trail Effect`，新圖形會依設定時間逐步消失
- 若關閉 `Trail Effect`，圖形會保留在畫面上，直到手動清除或擦除

## 10. 問題排除

### 啟動 `.app` 失敗

若直接打開 `build/MacDraw.app` 失敗，請改用：

```bash
./build/bin/MacDraw
```

### 無法點擊桌面或其他程式

代表目前仍在繪圖模式。

可使用：

- `Escape`
- `Disable Drawing`

### 圖形太快消失

請把 `Trail Disappear Time` 拉高，例如：

- `1.0 秒`
- `2.0 秒`
- `3.0 秒`

### 圖形不會消失

請確認：

- `Enable Trail Effect` 已勾選

## 11. 建議使用方式

如果你主要想做畫面標註，建議：

- `Shape` 用 `Rectangle`
- `Tool` 用 `Pen`
- `Trail Effect` 開啟
- `Trail Disappear Time` 設在 `0.5` 到 `1.5` 秒

如果你想做簡報式重點標示，這樣會比較俐落，不會讓畫面殘留太久。
